

# Carrossel de Carros

<img src="https://github.com/lucassantoos96/proj-fp-sellection/blob/main/assets/img-mockup-readme.png?raw=true"/>

## Descrição
Projeto front-end desenvolvido com HTML, CSS e JavaScript que apresenta um carrossel interativo com 5 modelos de carros diferentes.  
Cada slide exibe a imagem do carro e suas principais características, permitindo navegação suave entre os modelos.

## Funcionalidades

- Apresentação de  modelos de carros com imagens e informações

- Navegação manual e transição suave entre os slides 

- Layout estilizado com CSS para melhor experiência visual

## Tecnologias utilizadas

- HTML5

- CSS

- JavaScript
